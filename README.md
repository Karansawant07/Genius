## ⚙️ 使用说明

### 3. 设置您的账户
在使用 Genius 之前，您需要确保您的 TradeGenius 账户已经设置并且有足够的 USDC 或 USDT 余额。确保您已在 TradeGenius 上完成身份验证，以便可以进行交易。

### 4. 启动应用程序
1. 找到您下载的文件，并双击打开。
2. 应用程序将自动启动并要求您登录到您的 TradeGenius 账户。
3. 输入您的账户凭据并登录。

### 5. 自定义设置
在第一次运行中，您可以根据需要调整一些设置，例如选择要交易的代币数量，设置交易间隔等。应用程序会为您提供简单易懂的选项。

### 6. 开始交易
一旦设置完成，您可以点击“开始交易”按钮。Genius 将自动执行以下操作：
- 选择最优的代币进行交换。
- 在 Optimism 链上执行交易。
- 自动确认并持续交换，直到您选择停止。

### 7. 实时监控
Genius 提供实时日志和统计信息，帮助您监控您的交易活动。您可以随时查看当前余额和交易状态。
